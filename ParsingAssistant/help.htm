Главное меню

В момент вызова инструкций Вы находитесь в главном меню приложения.
Чтобы приступить к обучению, необходимо выбрать один из методов грамматического разбора для рассмотрения.

Всего доступно 4 метода для обучения:

  • левосторонний восходящий («слева направо»);
  • левосторонний нисходящий («сверху вниз»);
  • метод «сверху вниз» для LLk грамматик;
  • стековый метод для LRk грамматик.

Выбор осуществляется нажатием на соответствующую кнопку окна главного меню. Нажав на нее, Вы автоматически перейдёте в окно просмотра разбора.

Чтобы выйти из приложения, нажмите кнопку «Выход» или кнопку закрытия окна в правом верхнем углу экрана.

Окно просмотра разбора

Выбрав метод разбора нажатием на кнопку, Вы перешли в окно разбора.
В центральной части окна будет отображаться название метода грамматического разбора, выбранного в главном меню.
Для того, чтобы выбрать другой метод разбора, можно вернуться в главное меню приложения. Для этого необходимо нажать на кнопку «Назад» окна просмотра разбора.

  • Чтение теоретической информации о разборе
Вы можете начать обучение ознакомлением с теоретическими сведениями о выбранном методе разбора. Для этого нужно нажать на кнопку «Информация» окна разбора строки. Откроется окно, содержащее необходимую информацию о выбранном методе разора в формате текста.
  • Просмотр процесса разбора

Вы можете перейти в режим просмотра процесса разбора.
Для этого необходимо нажать на кнопку «Наглядный разбор». Окно разбора строки перейдет в состояние ожидания ввода строки для разбора.

Внешний вид окна разбора строки в состоянии просмотра разбора будет различен для разных методов разбора. Основное различие заключается в том, что для каждого из них будут отображены свои правила задания формального языка (секция правил), а также формат записей в таблице этапов разбора будет различен. Помимо этого для разбора стековым методом для LRk грамматик в секции правил будет отображена таблица предшествования операторов, используемая в процессе разбора.

  • Ввод строки

Приложение предоставляет Вам несколько способов ввода строки для проведения ее грамматического разбора.
Первый способ – ввод строки с клавиатуры. Вам требуется ввести строку вручную в поле для ввода.
Второй способ – генерация строки для разбора программой. Для того, чтобы сгенерировать строку, нужно выбрать и нажать любую из кнопок, находящихся в секции генерации строки. При этом по нажатию кнопки «Корректная» будет сгенерирована строка, разбор которой закончится распознанием конструкции языка, по нажатию кнопки «Некорректная» – строка, которая не будет распознана программой, а по нажатию кнопки «Случайная» – любая строка из двух, представленных выше.

Введя строку любым из возможных способов, Вы можете запустить процесс разбора, нажав на кнопку «Разобрать».
При попытке начать разбор с незаполненным полем входной строки будет показано соответствующее сообщение об ошибке. В этом случае программа продолжит ожидать ввода строки.

  • Режимы просмотра

После ввода строки и нажатия кнопки «Разобрать» окно разбора строки перейдет в состояние просмотра процесса разбора.
В момент начала просмотра в таблице хода разбора и в поле комментариев могут быть отображены записи, характеризующие начальные положения разбора. Они будут различаться для разных методов.

Для того, чтобы наблюдать процесс разбора по шагам, необходимо нажимать на кнопку «Шаг» с той периодичностью, какая будет комфортна для просмотра. С каждым нажатием на кнопку «Шаг» в поле комментариев будет появляться новая запись, характеризующая состояние процесса разбора.

Помимо этого могут происходить следующие события:

   - в таблице этапов разбора будут появляться строки, фиксирование которых важно для конкретного разбора, при этом строки могут быть выделены цветом для смысловой характеристики состояния разбора;
   - будет осуществляться подсветка правил (красным или зеленым цветом) в секции правил в момент рассмотрения конкретного правила в процессе разбора;
   - для стекового метода разбора в таблице предшествования операторов будет выделено рассматриваемое отношение операторов.

Порядок тех или иных событий определяется выбранным методом разбора и входной строкой, данной для разбора.
В момент, когда в таблице этапов разбора и в поле комментариев была выведена исчерпывающая информация о ходе разбора (т.е. когда разбор завершился), кнопка «Шаг» становится неактивной.

Помимо способа просмотра разбора по шагам Вам доступна возможность мгновенного получения всей информации о ходе разбора введенной строки. Для этого после перехода в состояние просмотра строки или в любой момент пошагового просмотра необходимо нажать на кнопку «Показать разбор полностью». В этом случае окно разбора строки сразу перейдет в состояние завершения просмотра.

  • Дополнительные возможности просмотра

После завершения просмотра разбора Вы можете посмотреть процесс разбора текущей введенной строки сначала. Для этого необходимо нажать кнопку «Просмотреть сначала». Окно разбора строки вновь перейдет в состояние начала просмотра разбора при текущей разбираемой строке.

В любой момент просмотра разбора (и после завершения просмотра) имеется возможность начать разбор другой строки выбранным методом. Для этого необходимо нажать на кнопку «Начать новый разбор». Окно разбора строки вновь перейдет в состояние ожидания ввода строки

Выход из приложения

Выход из приложения осуществляется нажатием на кнопку «Выход» главного меню или на крестик в правом верхнем углу окна главного меню или окна просмотра разбора.
